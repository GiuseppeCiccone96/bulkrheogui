# bulkrheogui
 Simple graphical user interface for analysis of bulk rheology data: 

<img src=images/maingui1.png  width=50% height=100%>
 
 ## supported files 
 Right now, data from the Anton Paar Rheometer (MCR 302) is supported. 

## requirements 
The following packages are required: `numpy`, `matplotlib`, `pandas`, `pyqt5`,`superqt`,`magicclass`.  It is suggested to create a virtual environment for the project either through pip or conda. 

## installation
The project is at a very early stage and no installer is provided. 
Please feel free to give feedback and sugeestions for features to add!
To launch the GUI, run `python main.py` from the command line.
